# **Roadmap \- Arcane**

## **Versão 0.1 \- Fundação**

Objetivo: estabelecer a base técnica do sistema.

### **Backend**

* Configurar FastAPI  
* Configurar MongoDB  
* Implementar arquitetura em camadas  
* Configurar sistema de autenticação

### **Frontend**

* Estruturar projeto React  
* Implementar sistema de login  
* Implementar layout principal

### **Infraestrutura**

* Docker  
* Variáveis de ambiente  
* Configuração inicial do projeto

---

## **Versão 0.2 \- Campanhas e Personagens**

Objetivo: permitir criação e gerenciamento de campanhas.

### **Funcionalidades**

* CRUD de campanhas  
* CRUD de personagens  
* Associação entre personagens e campanhas  
* Visualização de campanhas

---

## **Versão 0.3 \- Sessões Narrativas**

Objetivo: iniciar sessões jogáveis.

### **Funcionalidades**

* Criar sessão  
* Encerrar sessão  
* Histórico de sessões  
* Registro de mensagens  
* Registro de eventos

---

## **Versão 0.4 \- Integração com IA**

Objetivo: introduzir geração narrativa.

### **Funcionalidades**

* LLM Provider  
* Integração OpenAI  
* Integração Gemini  
* Interface para múltiplos provedores  
* GameMasterAgent

---

## **Versão 0.5 \- Sistema de Memória**

Objetivo: persistência narrativa.

### **Funcionalidades**

* MemoryService  
* MemoryAgent  
* Coleção Memories  
* Consolidação de eventos  
* Recuperação de contexto

---

## **Versão 0.6 \- Campanhas Persistentes**

Objetivo: continuidade entre sessões.

### **Funcionalidades**

* Resumo automático de sessões  
* Atualização de memórias  
* Contexto dinâmico  
* Continuidade narrativa

---

## **Versão 0.7 \- MVP Público**

Objetivo: primeira versão utilizável.

### **Funcionalidades**

* Criação completa de campanhas  
* Sessões jogáveis  
* Memória persistente  
* Integração com IA  
* Interface funcional

---
